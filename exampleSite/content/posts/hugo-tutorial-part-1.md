this is the first part of a series on getting started with hugo.

## what is hugo?

hugo is a fast and modern static site generator written in go. it's designed to make website creation fun again.

## installing hugo

you can install hugo using your package manager:

```bash
# macos
brew install hugo

# linux
sudo apt install hugo

# windows
choco install hugo
```

in the next part, we'll create our first hugo site.
